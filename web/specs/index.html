# Application overview

We will be developing an EAP (Employee Assistance Program) and Managed Care management software. We will be replacing the currently used software at Quantum, https://hartingeap.com/.

The application will contain:

  • dozens of screens / dialogs of form inputs modeling various data such as client companies, employees, providers, including many properties each.
  • a few highly interactive, frequently used screens that need to be optimized for speed of input and related to that performance of UI elements.
  • various documents printed as PDF files
  • a number of reports to be exported in various data formats such as CSV, PDF.
  • capability to import some data from CSV (for ex. employee demographics)

Principal users of the application will typically be a small number (<10) of internal staff that are each assigned to specific tasks / companies.

Providers will also be able to log into the software to perform various tasks related to service authorizations and payments.

Employers, employees and their family members (those that are receiving medical services) will not access the software.
